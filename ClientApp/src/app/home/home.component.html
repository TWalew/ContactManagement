Hello, Invelop!

Welcome to my single-page application, built with:

  • ASP.NET Core and C# for cross-platform server-side code
  • Angular and TypeScript for client-side code
  • Bootstrap for layout and styling
  • PrimeNG for UI
If you have configured and done all the steps from the Readme.md file in the Github repo you can skip these steps

To get started you'll have to run this query inside SQL Server Management Studio! I have used SQL Server for database!

First run this query to create the database:

Create Database ContactDB

After that run a query inside the database like so:

Create table dbo.Contacts( ContactID int identity(1,1) NOT NULL, FirstName varchar(500), Surname varchar(500), DateOfBirth date, PhoneNumber varchar(15), IBAN varchar(35), PRIMARY KEY (ContactId) )

After the database is created you can simply run this query to add the first contact. You can also skip that and add the first contact trough the front-end!

insert into dbo.Contacts values ('Toni', 'Valev', '1996-02-24','Stara Zagora, Avgusta Trajana 55', '0876876636', 'DE89370400440532013000')

After this initial setup you can simply enjoy the app, by running it directly from Visual Studio. It's configured to build and start the front-end immediately.
